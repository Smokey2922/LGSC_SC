## 🔍 What is LGSC?

LGSC is a **professional-grade cleanup tool** that goes far beyond what a standard uninstaller does.

When you uninstall an application normally, it leaves behind:
- Log files scattered across your system
- Registry entries and leftover folders
- Temporary files, crash reports, and hidden data
- Anti-cheat drivers and background processes

**LGSC finds all of it and removes it completely.**

## 🚀 Getting Started

### Requirements
- Windows 10 / 11
- Internet connection (for license activation)
- Discord account (for verification)

### Installation
1. Download the latest `Scanner.exe` from [Releases](https://github.com/Smokey2922/LGSC_SC/releases/latest)
2. Run the installer — no admin rights needed for basic scan
3. Enter your **activation code** (received from the owner)
4. Verify with **Discord**
5. Start scanning
